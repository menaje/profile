# project/docs 문서 구조 안내

## 폴더 분류

- `제안서/`
  외부 회사나 협업 파트너에게 전달할 수 있는 제안서 패키지
- `분석/`
  경력, 도메인, 역량, 성향 등 내부 분석 자료
- `기초자료/`
  PDF, README, 원본 이력서 등 참고용 기초 자료
- `작업계획/`
  문서 작성이나 산출물 생성 과정에서 사용한 작업 계획 문서

## 현재 제안서 패키지

- `제안서/건축사업_데이터_플랫폼/`
  건축사업 데이터 플랫폼 제안 문서 일체

## 분류 원칙

- 대외 제안 목적 문서는 `제안서`에 둔다.
- 개인 검토나 사고 정리 문서는 `분석`에 둔다.
- 원문 또는 참고 파일은 `기초자료`에 둔다.
- 작업 이력과 산출 계획은 `작업계획`에 둔다.
