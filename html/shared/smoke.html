WP-16 Shared CSS Smoke

한국어와 English가 함께 있는 A4 인쇄 smoke test

이 문서는 shared token, base, print CSS만으로 Pretendard와 Noto Sans CJK KR fallback, 18mm A4 여백, 배경 인쇄, keep-all 동작을 점검하기 위한 최소 검증 페이지입니다.

Pretendard 400/600/700 Noto Sans CJK KR fallback A4 18mm @media print

기준일은 2026-03-20이며, 이 smoke page는 실제 이력서 콘텐츠를 포함하지 않습니다. 대신 한국어 문장, Latin token, number, punctuation, 그리고 카드형 블록이 공통 CSS 위에서 안정적으로 보이는지만 확인합니다.

예시 문장: CONI shared layer keeps Korean copy readable while preserving color contrast, print background, and predictable page breaks for PDF export.

Verification Focus

  • 한글 glyph가 네모 tofu 없이 렌더링되는지
  • 한국어와 영문이 섞여도 줄바꿈이 과하게 깨지지 않는지
  • 카드 배경색과 gradient가 인쇄 미리보기에서 유지되는지
  • 두 번째 페이지로 넘어갈 때 page break가 안정적인지

Scope Boundary

WP-16은 구조와 shared layer만 다룹니다. resume, technical-career, personal-statement, haean-deck의 문서별 HTML 조판은 후속 WP에서 구현합니다.
