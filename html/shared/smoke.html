한국어와 English가 함께 있는 A4 인쇄 smoke test

이 문서는 cool-neutral 팔레트, 확대된 타이포 스케일, sharp 보더 반경, 차별화된 패널 프리미티브가 공통 CSS 위에서 안정적으로 동작하는지 점검하기 위한 최소 검증 페이지입니다.

Pretendard 400/600/700 Noto Sans CJK KR fallback A4 18mm compatibility layer enabled

기준일은 2026-03-21이며, 이 smoke page는 실제 이력서 콘텐츠를 포함하지 않습니다. 대신 한국어 문장, Latin token, number, punctuation, 그리고 새 shared component들이 공통 CSS 위에서 안정적으로 보이는지만 확인합니다.

예시 문장: the shared layer now separates base, print, and slide semantics so later document workplans can differentiate layout and accent without fighting a beige global skin.

Verification Focus

  • 한글 glyph가 네모 tofu 없이 렌더링되는지
  • tokens-base / tokens-print 분리가 print 문서에서 유지되는지
  • neutral panel primitives가 shared 기본값으로 동작하는지
  • 두 번째 페이지로 넘어갈 때 page break가 안정적인지

Scope Boundary

WP-33은 shared layer의 실제 색상값·타이포·반경·프리미티브를 전면 교체합니다. 문서별 레이아웃 적용은 후속 WP에서 구현합니다.
